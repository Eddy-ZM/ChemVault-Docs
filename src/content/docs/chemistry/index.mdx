## How to use this section

1. Start with the decision map if the mechanism is unclear.
2. Open the mechanism note for rate law, stereochemistry and substrate effects.
3. Use the reagent pages when the question is about scope or selectivity.
4. Use the troubleshooting page when observation and prediction do not match.
5. Check related pages at the bottom of each note before finalising an answer.
