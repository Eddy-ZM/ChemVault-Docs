# 产品地图

ChemVault 的产品可以分成四层：公开知识入口、科研工作台、账号和协作基础设施、Apple 原生 App。

## 公开知识入口

| 产品 | 入口 | 主要功能 |
| --- | --- | --- |
| ChemVault 主站 | [chemvault.science](https://chemvault.science) | 化学知识基础设施首页、化合物检索、研究方向、项目说明、公开数据和工具入口。 |
| ChemVault 中文站 | [chemvault-zh.pages.dev](https://chemvault-zh.pages.dev) | 中文内容展示、ICP备案准备、中文化功能页和公开说明。 |
| ChemVault Docs | [docs.chemvault.science](https://docs.chemvault.science) | 本手册、教程、说明、化学参考和课程知识库。 |

## 科研工作台

| 产品 | 推荐入口 | 适合任务 |
| --- | --- | --- |
| ChemVault Extract | [app.chemvault.science](https://app.chemvault.science) | 上传论文、报告、表格和仪器文件，运行 AI 提取，人工审核并导出结构化数据。 |
| ChemVault Files | [files.chemvault.science](https://files.chemvault.science) | 管理私有研究文件、文件夹、标签、权限、分享链接和文件处理状态。 |
| ChemVault Molecule Studio | [model.chemvault.science/molecule](https://model.chemvault.science/molecule) | 搜索分子、输入 SMILES、绘制结构、上传结构文件、加载 PDB 并查看三维结构。 |

## 账号和协作基础设施

| 产品 | 入口 | 主要功能 |
| --- | --- | --- |
| ChemVault User Center | [user.chemvault.science](https://user.chemvault.science) | 统一账号、资料、安全设置、计划状态、服务访问权限和管理员控制台。 |
| ChemVault Mail | [mail.chemvault.science](https://mail.chemvault.science) | 网页邮箱、写信、收件箱、星标、草稿、IMAP/SMTP、App Password 和邮件管理。 |
| ChemVault Notifications | [notify.chemvault.science](https://notify.chemvault.science) | 通知中心、项目消息、文件处理进度、AI 结果审核提醒、产品更新和管理员广播。 |

## Apple App

| App | 支持平台 | 主要功能 |
| --- | --- | --- |
| ChemVault Apple App | iPhone、iPad、Mac | ChemVault 模块导航、文档、AI 数据提取入口、文件、邮箱、通知和账号状态。 |
| ChemVault Mail Apple App | iPhone、iPad、Mac | 原生邮箱收件、阅读、写信、星标、账号设置和管理员入口。 |
| ChemVault Molecule Apple App | iOS 17+、iPadOS 17+、macOS 14+ | 分子搜索、SMILES、基础绘制、PDB、本地分子库、三维查看和 XYZ 导出。 |

## 产品之间的关系

- **User Center 是身份层**：登录、资料、安全、服务权限和管理员操作都从这里统一。
- **Files 是文件层**：谱图、数据集、论文、手稿、实验记录和项目材料放在文件系统里。
- **Extract 是数据提取层**：把论文、报告和表格转成可审核的结构化记录。
- **Notifications 是协作层**：文件处理、提取任务、审核结果、项目消息和产品更新都在这里通知。
- **Mail 是沟通层**：个人邮箱、团队邮箱和第三方邮件客户端配置都在这里。
- **Molecule Studio 是结构查看层**：处理 SMILES、PDB、结构文件和三维可视化。
- **主站和 Docs 是公开说明层**：对外解释功能、教程、知识记录和项目边界。
