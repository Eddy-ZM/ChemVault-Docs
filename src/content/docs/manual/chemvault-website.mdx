ChemVault 主站和中文站是公开入口，用来展示化学知识基础设施、研究方向、项目说明、工具入口和公开记录。它们适合浏览、检索、学习和对外介绍。

## 入口

- 英文主站：[https://chemvault.science](https://chemvault.science)
- 中文站：[https://chemvault-zh.pages.dev](https://chemvault-zh.pages.dev)
- 手册站：[https://docs.chemvault.science](https://docs.chemvault.science)

## 如何检索化合物

1. 打开主站或中文站首页。
2. 点击 **Compounds / 化合物**。
3. 输入化合物名称、分子式、功能团、反应意图或标识符。
4. 使用筛选项缩小范围，例如 record type、domain、topic 或本地高级筛选。
5. 打开结果页查看结构化摘要、记录类型、相关链接和研究上下文。

## 典型检索词

- `aspirin`
- `benzene`
- `ketone to alcohol`
- `aryl chloride`
- `sulfonamide`
- `PCC`
- `NaBH4`

## 搜索建议

| 目标 | 推荐关键词 |
| --- | --- |
| 找具体化合物 | 英文名、常用名、分子式、CID。 |
| 找反应方向 | `ketone to alcohol`、`aryl chloride coupling`。 |
| 找试剂用途 | `PCC`、`NaBH4`、`LiAlH4`。 |
| 找官能团相关内容 | `sulfonamide`、`carbonyl`、`alkene`。 |
| 找课程或参考内容 | 使用 Docs 搜索框检索中文或英文关键词。 |

## 公开站点与登录产品的区别

| 内容 | 公开站点 | 登录产品 |
| --- | --- | --- |
| 化合物和公开记录 | 可浏览。 | 可与私人文件和项目结合。 |
| 教程和参考 | 可浏览。 | 可能关联个人账号状态。 |
| 私有文件 | 不展示。 | 在 Files 中管理。 |
| AI 提取 | 只介绍功能。 | 在 Extract 中运行和审核。 |
| 邮箱、通知、权限 | 只提供入口。 | 在 Mail、Notifications、User Center 中使用。 |

## 中文站的定位

中文站是主站的中文静态版本，面向中文内容展示、中文化功能页和 ICP 备案审核准备。化学名称、缩写、外部数据库名和部分专业术语会保留英文，以保持学术检索准确。

## 公开数据

主站和中文站都包含公开数据说明和 `public-record-index.json` 相关入口。公开数据适合用于浏览、索引和参考，不应被理解为实验、安全、医疗或商业建议。

## 引用和使用建议

- 公开记录适合作为学习、检索和项目导航材料。
- 需要正式科研数据时，应回到原始文献、Files 原始文件或 Extract 的 evidence-backed records。
- 中文站中的英文术语通常用于保持检索准确性，不应强行翻译。
- 涉及安全、毒性、医学或商业决策时，应使用权威来源和人工判断。
