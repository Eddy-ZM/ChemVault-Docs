## 链接维护规则

- 生产链接只写用户可访问的站点和 App。
- API、Webhook、SDK 等开发链接应放在 Extract 或 Notifications 教程中，不放在新用户入口首页。
- 如果一个产品仍在域名迁移中，教程里应明确推荐入口，并在相关产品页说明可能出现的旧入口。
