ChemVault 生态包含多个 Apple 设备入口。不同 App 面向不同任务：主 ChemVault App 用于模块导航，Mail App 用于邮箱，Molecule App 用于分子结构查看。

## ChemVault Apple App

支持平台：

- iPhone
- iPad
- macOS

主要模块：

- Home：可用模块和工作区状态。
- Molecular Modelling：分子搜索、SMILES、结构预览和建模入口。
- Scientific File Storage：研究文件、仪器输出和项目材料。
- Documentation：文档、工作流笔记和参考材料。
- AI Scientific Data Extraction：论文、PDF 和仪器文件的结构化提取入口。
- ChemVault Mail：项目通信和研究邮件。
- Account and Permissions：账号状态、模块访问和权限可见性。
- Notifications：提取任务、部署和文件处理事件。
- Settings：语言、区域、外观和连接偏好。

## 主 App 使用场景

| 场景 | 推荐入口 |
| --- | --- |
| 查看自己能用哪些模块 | Home / Account and Permissions |
| 快速进入分子工具 | Molecular Modelling |
| 查看文档或教程 | Documentation |
| 从移动端打开文件入口 | Scientific File Storage |
| 查看任务和事件 | Notifications |
| 修改语言或外观 | Settings |

## ChemVault Mail Apple App

支持平台：

- iPhone
- iPad
- Mac

主要功能：

- 收件箱。
- 邮件阅读。
- 写信。
- 星标邮件。
- 刷新、标记已读、星标、删除。
- 登录和注册。
- 邮箱地址管理。
- 个人设置。
- 密码修改入口。
- 管理功能入口。

管理员可访问：

- 用户管理。
- 角色管理。
- 注册邀请码管理。
- 全部邮件管理。
- 系统设置。
- 数据分析。

体验特性：

- 自适应布局。
- 深色模式。
- 公告与维护提示。
- 权限可见性。

## Mail App 操作建议

- 处理短邮件、查看星标和快速回复适合移动端。
- 带多个附件、正式长邮件或管理员批量操作优先使用桌面端。
- 第三方客户端配置仍应使用 App Password。
- 如果 App 进入维护模式，应以网页 Mail 和 User Center 状态为准。

## ChemVault Molecule Apple App

支持平台：

- iOS 17+
- iPadOS 17+
- macOS 14+

主入口：

- Search
- SMILES
- Draw
- PDB
- Library
- Account

### Search

支持名称和 PubChem CID 搜索。常用示例包括 Water、Ethanol、Benzene、Caffeine、Aspirin、Paracetamol、Ibuprofen、Glucose。

### SMILES

支持多行 SMILES 输入、常用示例、Load Molecule 和 Clear。

### Draw

支持基础原生绘图：

- 放置原子。
- 创建单键。
- 选择、擦除、撤销、重做、清空。
- 36 个常用元素。
- 为简单绘制结构生成 SMILES。

### PDB

支持四位 PDB ID，例如 `1CRN`、`4HHB`、`1BNA`。

### 分子详情页

支持：

- 三维分子查看。
- Ball and stick、Sphere、Stick。
- 背景切换。
- 显示名称、分子式、分子量、Canonical SMILES、InChIKey、IUPAC Name、来源、PDB ID 和文件名。
- 保存到本地 Library。
- 分享结构摘要。
- 复制 SMILES。
- 导出 XYZ 文本。

### Library

本地分子库支持保存、查看、删除和导入 `.mol`、`.sdf`、`.xyz`、`.pdb`、`.smi`、`.smiles`、`.txt`。

## Molecule App 与网站端区别

| 能力 | Apple App | 网站端 |
| --- | --- | --- |
| 移动端快速查询 | 更适合。 | 可用但屏幕空间较小。 |
| 本地 Library | 更适合随手保存。 | 更适合导出和桌面整理。 |
| 复杂文件上传 | 可导入常见结构文件。 | 更适合处理较复杂文件和桌面下载。 |
| 报告截图 | 可分享结构摘要。 | 更适合导出 PNG。 |
| 批量科研流程 | 不推荐。 | 使用 Files、Extract 和 Docs。 |

## 使用建议

- 移动端查询分子：使用 Molecule App。
- 移动端处理邮件：使用 Mail App。
- 查看模块和账号权限：使用主 ChemVault App 或 User Center。
- 复杂批量提取和审核：优先使用桌面浏览器中的 Extract。

## 登录和权限提示

- 如果 App 中某个模块不可见，先在 User Center 检查 Service Access。
- 如果管理员入口不可见，检查 Page Access 或角色。
- 如果 Mail App 不能收发信，先确认网页 Mail 是否可用。
- 如果 Molecule App 搜索失败，改用 SMILES、PDB ID 或稍后重试。
