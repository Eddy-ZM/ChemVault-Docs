ChemVault Mail 是 ChemVault 生态的邮箱服务，提供网页邮箱和 Apple 原生 App 两类入口。它适合个人邮箱、团队邮箱、管理员运维和移动端访问。

## 入口

- Web Mail：[https://mail.chemvault.science](https://mail.chemvault.science)
- 账号安全和主账户管理：[https://user.chemvault.science](https://user.chemvault.science)

## 邮箱使用

### 收件箱

收件箱用于查看收到的邮件、未读状态、邮件时间、发件人、主题和正文预览。

建议：

- 优先处理 unread。
- 使用星标标记重要邮件。
- 对项目邮件使用明确主题，便于后续检索。

### 邮件详情

邮件详情页用于阅读完整内容，查看 HTML 邮件、附件和收件地址。遇到外部链接或附件时，应先确认来源。

### 写信

写信流程：

1. 选择发件邮箱。
2. 填写 To、Cc、Bcc。
3. 填写 subject。
4. 编写正文。
5. 检查附件和收件人。
6. 发送。

### 草稿

草稿用于保存未完成邮件。长邮件、正式通知或带附件的邮件建议先保存草稿再发送。

### 星标邮件

星标用于集中管理重要邮件，例如待回复、合同、账号通知、关键项目沟通。

## 多邮箱地址

同一个用户可以管理多个邮箱地址，适合：

- 个人邮箱。
- 项目别名。
- 团队收件地址。
- 不同用途的邮箱身份。

## 邮件组织建议

| 邮件类型 | 建议做法 |
| --- | --- |
| 项目沟通 | 主题包含项目名、样品或任务编号。 |
| 文件附件 | 下载后放入 Files，并在邮件中保留来源。 |
| 审核结果 | 用简短结论加导出文件或数据集链接。 |
| 系统通知 | 星标重要账号、权限、账单或安全邮件。 |
| 长期任务 | 先保存草稿，确认收件人和附件后再发送。 |

## 账户与安全

| 功能 | 说明 |
| --- | --- |
| 登录与注册 | 支持 ChemVault Mail 账号登录、注册和邀请码流程。 |
| 账户资料 | 查看邮箱、用户名和角色。 |
| 主账户管理 | 密码和账号安全通过 User Center 统一处理。 |
| 语言设置 | 支持中文和英文切换。 |
| 账户删除 | 在权限允许时可以删除自己的账号。 |

## IMAP/SMTP 和 App Password

第三方客户端如 Outlook、Apple Mail 需要 IMAP/SMTP 配置。为了安全，不应直接使用主账号密码，而应使用 App Password。

使用建议：

1. 在 Mail 设置里查看客户端配置。
2. 创建 App Password。
3. 创建后立即保存，因为原始密码通常只展示一次。
4. 如果设备丢失或不再使用，撤销对应 App Password。

## 客户端配置检查

| 项目 | 应检查 |
| --- | --- |
| 邮箱地址 | 是否使用完整地址。 |
| 认证密码 | 是否使用 App Password。 |
| IMAP | 用于收信，检查服务器、端口和安全方式。 |
| SMTP | 用于发信，检查服务器、端口、认证和安全方式。 |
| 发件身份 | 是否选择正确邮箱地址或别名。 |
| 撤销 | 设备停用后是否撤销对应 App Password。 |

## 管理功能

管理员可以管理：

- 用户列表、搜索、筛选、创建、禁用、启用和恢复。
- 邮箱地址和用户绑定关系。
- 角色与权限。
- 注册邀请码和使用记录。
- 全部邮件数据。
- 系统设置，包括网站标题、注册开关、收发信、域名策略、转发规则、黑名单、对象存储、通知和外观。
- Apple App 配置，包括版本提示、维护模式、公告、主题、链接和资源。

## 数据看板

数据看板用于判断系统运行情况：

- 总收件数。
- 总发件数。
- 邮箱地址数。
- 用户数。
- 用户增长。
- 收发邮件增长。
- 每日发送量。
- 邮件来源分布。
- 正常数据和已删除数据拆分。

## 管理员日常检查

- 是否有长期未登录但仍保留邮箱地址的账号。
- 是否有异常增长的发信量。
- 邀请码是否仍在有效期内。
- 系统设置是否允许注册、发信和收信。
- 黑名单、转发规则和对象存储设置是否符合当前策略。
- Apple App 维护模式和公告是否需要更新。

## Apple App

Mail Apple App 支持 iPhone、iPad 和 Mac 的邮箱访问与管理。

主要功能：

- 收件箱。
- 邮件阅读。
- 写信。
- 星标邮件。
- 刷新、标记已读、星标、删除。
- 登录和注册。
- 邮箱地址查看和管理。
- 个人设置。
- 密码修改入口。
- 具备权限时访问用户管理、角色管理、邀请码、全部邮件、系统设置和数据分析。
