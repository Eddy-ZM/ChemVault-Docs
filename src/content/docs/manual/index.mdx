ChemVault 不是单个页面，而是一组围绕化学知识、科研文件、AI 数据提取、账号权限、邮箱、通知、分子可视化、品牌素材和传播内容的产品与资料集合。这个手册把各个产品整理成可学习、可查阅、可交接的教程，并把 `E:\CV_suite` 中相关内容映射到对应站点和说明页。

## 适用人群

- **新用户**：想知道从哪里开始、每个站点负责什么。
- **学生和课程用户**：需要检索化合物、查看试剂、阅读课程资料或准备实验。
- **研究者**：需要整理论文、实验报告、谱图、材料记录和结构化数据。
- **团队管理员**：需要管理账号、权限、邮箱、通知、文件访问和审计。
- **开发或集成用户**：需要了解 API Key、Webhook、导出数据和相关端点。

## 推荐学习路径

新用户建议按“入口、身份、文件、数据、协作”的顺序阅读：

1. 先用 [产品地图](/manual/product-map/) 建立全局概念。
2. 如果需要核对本机目录、内容资产或传播素材，阅读 [Suite 内容索引](/manual/suite-content-index/)。
3. 再用 [快速开始](/manual/getting-started/) 按角色选择路线。
4. 如果涉及团队或权限，阅读 [角色与权限教程](/manual/roles-permissions/)。
5. 如果涉及科研文件，阅读 [Files 教程](/manual/files/) 和 [Extract 教程](/manual/extract/)。
6. 如果需要交付数据，阅读 [数据生命周期教程](/manual/data-lifecycle/)。
7. 遇到问题时回到 [FAQ 与排障](/manual/faq-troubleshooting/)。

## 如何使用这份手册

1. 如果你不知道要打开哪个站点，先看 [产品地图](/manual/product-map/)。
2. 如果你知道自己的任务，例如上传论文、发送邮件或检索化合物，直接进入对应教程。
3. 如果遇到英文缩写或专业词，使用 [术语表](/manual/glossary/)。
4. 如果你在维护 `E:\CV_suite` 中的内容，使用 [Suite 内容索引](/manual/suite-content-index/) 找到对应教程和生产入口。
5. 如果需要跳转生产站点，使用 [相关链接](/manual/links/)。

## 内容边界

手册聚焦用户能看到和能操作的内容：页面入口、按钮含义、功能用途、角色权限、工作流顺序和术语解释。它不包含代码结构、数据库设计、部署命令、密钥配置或实现原理。
