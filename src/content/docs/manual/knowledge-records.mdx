ChemVault 的公开知识页面围绕 chemical records 和 research context 组织。你可以把它理解为一套面向化学学习、科研整理和知识检索的结构化目录。

## Compound Search

Compound Search 用于搜索化合物、分子式、标识符、本地记录和研究主题。

使用方式：

1. 进入主站 **Compounds** 页面。
2. 输入名称、分子式、SMILES、反应意图或关键词。
3. 通过主要筛选和高级筛选收窄结果。
4. 在结果页切换本地记录、academic search 或 session imports。
5. 打开记录详情，继续查看关联试剂、材料、方法或图谱。

## 记录阅读顺序

打开一条知识记录后，建议按以下顺序阅读：

1. 先看标题、record type 和简短摘要。
2. 再看结构、公式、标识符或主题分类。
3. 查看 linked context，理解它和试剂、方法、谱图或 Dossier 的关系。
4. 阅读 evidence 或 source 相关说明。
5. 最后根据任务跳转到 Files、Extract、Molecule Studio 或 Docs 参考页面。

## Research Workbench

Research Workbench 是研究型记录查看区，用于从多个角度阅读同一主题。

常见视角：

- Reaction systems：反应体系和相关条件。
- Evidence graph：记录之间的证据关系。
- Research lenses：用不同研究视角切换记录。
- Linked context：相关化合物、方法、试剂和来源。

## Reagents

Reagents 页面用于浏览试剂和转化记录。适合回答“这个试剂通常用于什么”“限制是什么”“和哪些反应相关”。

阅读建议：

- 先看试剂用途和 scope。
- 再看典型底物、条件、限制和注意事项。
- 最后跳转到相关反应或方法页。

## Materials

Materials 页面聚焦材料记录和材料科学上下文。适合整理材料属性、应用方向、来源信息和相关论文线索。

## Methods

Methods 页面用于查看方法、协议、证据标准和可重复性说明。它比普通笔记更强调 workflow、review、reproducibility 和 evidence。

阅读建议：

1. 先看方法目标。
2. 再看输入材料、输出结果和判断标准。
3. 如果需要操作文件或数据，转到 Files 或 Extract。

## Spectroscopy Workbench

Spectroscopy Workbench 用于结构证明和谱学推理记录。

适合场景：

- 查看 NMR、IR、MS、UV-vis 等证据说明。
- 对比结构和光谱证据。
- 整理“为什么这个谱图支持这个结构”的推理链。

## Mechanism Atlas

Mechanism Atlas 将反应机制作为 graph 来组织，关注 substrate assumptions、rate-law clues、stereochemical consequences 和 failure modes。

使用方式：

1. 从 Atlas 页面进入机制图谱。
2. 查看节点之间的反应和证据关系。
3. 结合 Workbench 或 Reagents 阅读上下文。

## Dossiers

Dossiers 是结构化研究档案。它适合把一个主题、一个项目或一个结论相关的证据集中展示。

常见内容：

- claim 或 research question
- supporting records
- evidence summary
- related compounds
- related methods

## Library

Library 是学术概念和来源感知参考材料的集合。它适合查术语、背景、概念关系和文献上下文。

## Public Data

Public Data 提供公开记录索引的说明和下载入口。它适合做浏览和索引参考，不等于完整实验数据库。

## 适合与不适合

| 适合 | 不适合 |
| --- | --- |
| 学习化学概念。 | 直接替代原始文献。 |
| 查找化合物和相关主题。 | 作为安全操作标准。 |
| 建立研究主题地图。 | 作为医疗、法律或商业承诺。 |
| 找到进入 Files、Extract 或 Molecule Studio 的线索。 | 代替人工审核的科研数据。 |

## 与 Molecule Studio 的配合

- 如果记录中有 SMILES、InChIKey 或 CID，可以复制到 Molecule Studio。
- 如果需要报告截图，使用 3D Viewer 导出 PNG。
- 如果需要保存结构文件，导出 SDF、Molfile、XYZ 或 PDB 后放入 Files。
