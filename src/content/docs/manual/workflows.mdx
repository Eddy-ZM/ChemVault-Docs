ChemVault 的价值来自产品之间的组合。下面是最常见的端到端路径。

## 从文献到结构化数据

1. 在主站或中文站搜索相关化合物、试剂和方法，确认主题背景。
2. 在 Files 上传论文、支持信息、CSV、XLSX 或实验报告。
3. 在 Extract 创建项目并上传需要解析的文档。
4. 等待 parse job 完成。
5. 估算 AI extraction 成本。
6. 运行 AI extraction。
7. 在 Notifications 查看任务进度和完成提醒。
8. 在 Review Workflow 中审核每条候选记录。
9. 批准可信记录，拒绝或修正问题记录。
10. 导出 CSV、JSON 或 XLSX。

交付前检查：

- 原始文件已保存在 Files。
- approved records 保留证据链。
- 导出文件命名包含项目和日期。
- 关键结论可追溯到文档页码和 evidence quote。

## 从谱图文件到项目归档

1. 在 Files 创建项目文件夹。
2. 上传 NMR、IR、MS、HPLC 或其他仪器输出。
3. 添加标签，例如 `NMR`、`raw`、`review`、`released`。
4. 在 Inspector 查看元数据和预览。
5. 对需要处理的文件进入 Review Flow。
6. 需要协作时创建只读 share。
7. 在 Notifications 跟踪文件处理事件。
8. 在 Dossiers 或 Methods 中记录结论和证据关系。

推荐标签：

- `raw`
- `processed`
- `NMR`
- `IR`
- `MS`
- `HPLC`
- `review`
- `released`

## 从分子想法到三维结构截图

1. 打开 Molecule Studio。
2. 用 Search、SMILES、Draw、Upload 或 PDB 加载结构。
3. 在 3D Viewer 中选择显示模式。
4. 调整背景、氢原子和标签。
5. 查看 Structure Details。
6. 导出 SMILES、SDF、XYZ、PDB 或 PNG。
7. 将导出的图片或结构文件放入 Files。

## 从项目消息到审核行动

1. 项目成员在 Notifications 中收到项目消息或任务提醒。
2. 打开相关 project conversation。
3. 查看文件、任务或结果详情。
4. 对提取结果执行 accept、reject、correct 或 unsure。
5. 审核完成后，生成 dataset 或导出文件。
6. 必要时通过 Mail 发送正式沟通。

## 从账号到权限开通

1. 用户在 User Center 注册或登录。
2. 完善 Profile。
3. 在 Security 绑定第三方账号或更新密码。
4. 管理员在 User Management 中查看用户。
5. 在 Service Access 中授权 Extract、Files、Mail、Notifications 或 Molecule。
6. 在 Page Access 或 Direct Permissions 中处理特殊权限。
7. 用户重新打开目标产品确认访问状态。

权限交接时应记录：

- 谁申请权限。
- 为什么需要权限。
- 开通哪些服务。
- 是否包含管理页或审核页。
- 何时复查或撤销。

## 从邮件沟通到文件和任务

1. 在 ChemVault Mail 收到合作方、课程或项目邮件。
2. 将重要邮件星标或整理到对应邮箱地址。
3. 下载附件后上传到 Files。
4. 如果附件是论文、报告或表格，进入 Extract 处理。
5. 在 Notifications 跟踪处理和审核状态。
6. 审核完成后，通过 Mail 回复结果或发送导出数据。

## 管理员维护闭环

1. User Center 检查用户和服务访问。
2. Files 检查文件权限、分享和异常上传。
3. Extract 检查 API Key、批量任务、计划限制和导出。
4. Notifications 检查 Webhook、广播、产品更新和审计日志。
5. Mail 检查用户、邮箱地址、邀请码、系统设置和数据看板。
6. 定期撤销不再需要的 key、share、App Password 和特殊权限。

## 工作流选择表

| 目标 | 最短路径 |
| --- | --- |
| 公开检索一个化合物 | 主站 → Compound Search → Molecule Studio |
| 保存项目原始资料 | Files → Folder/Tags → Inspector |
| 把论文提取成表格 | Files → Extract → Review → Export |
| 审核 AI 结果 | Notifications → Review item → Extract Review Workflow |
| 给合作方发送结果 | Extract Export → Files 归档 → Mail |
| 开通新成员 | User Center → Service Access → Page Access |
| 排查任务失败 | Notifications → 任务详情 → FAQ 与排障 |
