这个页面把 `E:\CV_suite` 下涉及 ChemVault 的网站、App、内容资产和维护资料整理成一个链接地图。生产用户应使用生产入口和站内教程；本地来源目录只用于维护者理解内容来源，不代表网页用户能直接访问。

## 阅读规则

- **生产入口** 是用户实际打开的网站、App 或 API。
- **站内教程** 是本手册里解释该产品如何使用的页面。
- **来源目录** 是维护者在 `E:\CV_suite` 下定位素材和功能说明时使用的目录。
- **不公开内容** 只说明边界，不在公开教程中展开密钥、环境变量、部署细节或私有凭据。

## 不应公开链接的内容

- `.env*`、API keys、App Store Connect private keys、Webhook secrets、数据库连接字符串和其他私有凭据。
- `_git_bundles` 中的 Git 历史 bundle，只用于本地恢复。
- `node_modules`、`.wrangler`、`.astro`、`dist`、`out`、`renders` 等依赖、缓存或构建产物。
- 包含完整用户数据、私有邮件、私有实验文件或未发布业务配置的本地资料。

## 维护规则

1. 新增产品时，先在本页登记来源目录、生产入口和站内教程。
2. 新增公开站点时，同步更新 [产品地图](/manual/product-map/) 和 [相关链接](/manual/links/)。
3. 新增 API、Webhook 或 SDK 文档时，优先链接到 [Extract](/manual/extract/) 或 [Notifications](/manual/notifications/)。
4. 新增账号、权限或管理员功能时，同步更新 [User Center](/manual/user-center/) 和 [角色与权限](/manual/roles-permissions/)。
5. 涉及密钥、部署、数据库或实现细节的内容只保留维护说明，不写入用户教程。
