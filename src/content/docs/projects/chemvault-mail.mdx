ChemVault Mail is the email system built for the ChemVault domain.

It is part of the ChemVault ecosystem and is designed to provide a professional communication layer for the project.

## Purpose

ChemVault Mail is used to explore:

- custom domain email
- email interface design
- Cloudflare-based web applications
- account and message management
- professional project branding

## Relationship with ChemVault Core

ChemVault Mail is separate from ChemVault Core.

ChemVault Core focuses on chemistry knowledge and research records.

ChemVault Mail focuses on communication and system identity.

## Future direction

ChemVault Mail may remain as an internal tool or portfolio project before being considered for wider public use.
