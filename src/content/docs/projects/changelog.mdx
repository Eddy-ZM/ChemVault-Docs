This changelog records meaningful public-facing changes to the documentation system.

## 2026-06-19

### SaaS information architecture upgrade

- Reworked the homepage into a ChemVault Knowledge OS product surface.
- Added a logical workflow strip: Capture, Structure, Decide and Publish.
- Added product suites, workflow switcher, system layers and feature matrix sections.
- Added Workspace Map, SaaS Operating Model and Feature Matrix pages.
- Expanded active records from 20 to 23 indexed routes.

## 2026-06-18

### Commercial navigation upgrade

- Reworked the homepage into a commercial command center for authority, utility and operations.
- Added audience lanes for students, lab work and product operations.
- Added commercial overview, roadmap, changelog and resource library routes.
- Expanded active library records so important operational pages are discoverable from the homepage.
- Added richer SEO and social metadata for the public documentation site.

### Interaction and animation layer

- Added homepage record search, workspace filters and empty-state feedback.
- Added scroll progress, reveal animations, pointer glow and a back-to-top control.
- Added keyboard focus support for the active-record search field with `/`.

## 2026-06

### Knowledge-library expansion

- Added structured chemistry routes for substitution, elimination, troubleshooting and carbonyl reductions.
- Added reagent passports for sodium borohydride, lithium aluminium hydride and PCC.
- Added lab guides for TLC monitoring and work-up/extraction planning.
- Added project manuals for ChemVault Core, ChemVault Mail and documentation maintenance.
