The Feature Matrix defines what the public documentation system already supports and what should be treated as roadmap-backed work.

## Capability matrix

| Capability | Current state | User value | Route |
| --- | --- | --- | --- |
| Global documentation search | Live | Find mechanisms, reagents, workflows and project pages quickly. | [Docs Home](/) |
| Active records filter | Live | Filter indexed records by workspace and status. | [Active Records](/#active-library-records) |
| Workspace routing | Live | Start from user jobs instead of folder structure. | [Workspace Map](/getting-started/workspace-map/) |
| Decision maps | Live | Choose reaction or reagent routes with less ambiguity. | [Reagent Selection Map](/reagents/reagent-selection-map/) |
| Resource release states | Planned | Prepare public assets with metadata and lifecycle status. | [Resource Library](/downloads/resources/) |
| Roadmap and changelog | Live | Show that the public product surface is maintained. | [Roadmap](/projects/roadmap/) |
| Documentation maintenance | Live | Keep sidebar, links, search and build output reliable. | [Docs Maintenance](/projects/chemvault-docs-maintenance/) |
| SaaS operating model | Live | Explain the product layers and commercial quality bar. | [SaaS Operating Model](/projects/saas-operating-model/) |

## Readiness labels

- **Live** means the route exists and users can use it now.
- **Ready** means the content is usable but not necessarily a product-level system.
- **Planned** means the structure exists, but public assets or automation are not fully released.
- **Indexed** means the content is present in the documentation/search layer.

## What not to fake

Do not add pricing, testimonials, enterprise security claims, analytics dashboards or customer logos until those surfaces are real. Commercial polish should come from clear structure, trustworthy maintenance and useful workflows.

## Next feature candidates

- Resource cards with file type, audience, owner and release state.
- More record metadata for owner, version and last reviewed date.
- Curated revision packs for CHEM10712.
- A deeper taxonomy for mechanisms, reagents, lab workflows and resource types.
