ChemVault is an independent chemistry knowledge platform developed by Ziwen Mu.

It aims to organise chemistry notes, reagent references, reaction mechanisms, laboratory guides, and project documentation into a structured digital archive.

## ChemVault ecosystem

- [**ChemVault Core**](/projects/chemvault-core/) — the main chemistry research platform.
- [**ChemVault Mail**](/projects/chemvault-mail/) — the email system for the ChemVault domain.
- [**ChemVault Docs**](/projects/chemvault-docs-maintenance/) — the documentation and knowledge base.
- [**ChemVault Commercial Overview**](/projects/commercial-overview/) — the public positioning and operating model.
- [**ChemVault Files**](/downloads/resources/) — the resource and download planning area.

## Purpose

ChemVault is designed to support chemistry learning, laboratory preparation, project documentation, and long-term scientific knowledge management.
