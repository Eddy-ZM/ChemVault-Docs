The Workspace Map is the first routing layer of ChemVault Docs. It helps users decide where to start before they open individual chemistry, lab, resource or product pages.

## Start with the job

| User job | Start here | Expected output |
| --- | --- | --- |
| Understand a concept | [Chemistry Notes](/chemistry/) | A mechanism, map or troubleshooting route |
| Choose a reagent | [Reagent Selection Map](/reagents/reagent-selection-map/) | A reagent shortlist with scope and risk checks |
| Prepare lab work | [Lab Guides](/lab-guides/) | A monitoring or work-up workflow |
| Revise CHEM10712 | [CHEM10712 Index](/uom-chemistry/chem10712/) | A course topic path or answer pattern |
| Publish a resource | [Resource Library](/downloads/resources/) | A release state and metadata checklist |
| Maintain the product surface | [SaaS Operating Model](/projects/saas-operating-model/) | A route, status and changelog decision |

## Routing rules

- Start from the user job before choosing a section.
- Prefer decision maps when a user is choosing between alternatives.
- Prefer lab guides when the output must be an action sequence.
- Prefer project manuals when the output is governance, release or maintenance.
- Link every resource candidate to a context page before public release.
