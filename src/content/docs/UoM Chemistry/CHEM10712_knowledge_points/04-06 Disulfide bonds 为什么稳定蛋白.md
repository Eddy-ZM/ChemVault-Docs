## 详细内容

disulfide bond 的稳定作用不只是多了一根 covalent bond，更重要是降低 unfolded state 的 conformational entropy。

规则：

- Cys 在序列上相距越远，但三维结构中越近，形成 disulfide 后对 unfolded chain 的限制越大。
- 长程 disulfide 更能降低 unfolded state 的自由度，因此稳定性提升更明显。
- extracellular proteins 常富含 disulfides，因为胞外环境更 oxidising、pH/盐/温度更变化。

## 深入讲解

disulfide bond 稳定蛋白的主要热力学原因是降低 unfolded state 的 entropy。未形成 S-S 时，展开链可以探索大量构象；形成一个 long-range disulfide 后，两个远距离序列位置被拴在一起，展开态可选构象大幅减少，因此 unfolded state 被相对 destabilized，folded state 看起来更稳定。

为什么胞外蛋白常有 disulfides？胞外环境更 oxidising、变化更大，蛋白需要长期稳定；胞内环境更 reducing，disulfide 不易稳定存在，而且细胞内很多蛋白依赖 dynamics 和 chaperones。

- **原文小点：** Cys 在序列上相距越远，但三维结构中越近，形成 disulfide 后对 unfolded chain 的限制越大。
  - **讲解：** 这个小点是在说明 disulfide stabilization；核心不是多一根键，而是限制 unfolded state conformational entropy。

## 做题和复习时怎么用

若题目问哪一个 disulfide 更稳定，优先选序列距离远但三维距离近的 Cys pair，因为它更大幅限制 unfolded chain entropy。

## 常见误区

- 把 protein structure 当成静态不动的模型。
- 只说“形成更多键所以稳定”，不讨论 unfolded state entropy 或 dynamics。
