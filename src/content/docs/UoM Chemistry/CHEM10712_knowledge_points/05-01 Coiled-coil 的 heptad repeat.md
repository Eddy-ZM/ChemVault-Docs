# 5.1 Coiled-coil 的 heptad repeat

## 详细内容

coiled-coil 常由 heptapeptide motif `abcdefg` 重复形成。由于 alpha helix 每圈约 3.5 个 residues，7 个 residues 约对应两圈，因此 a/d 位点可周期性朝同一侧形成 hydrophobic interface。

常见规律：

- a 和 d 位点常为 hydrophobic residues。
- e/g 位点常参与 charged/polar interactions，帮助决定 parallel 或 anti-parallel directionality。
- coiled-coil 例子包括 keratin、leucine zipper、tropomyosin/actin complex。

工程化思路：

- 设计 hydrophobic interface 让两条 helix 配对。
- 用 charged residues 设计方向性。
- 加入 Cys 形成 disulfide bond 可提高 thermostability。
- 也可通过改变 buffer、salt、co-solute 等非序列方式稳定。

## 深入讲解

coiled-coil 是理解 sequence-to-structure 的好例子，因为 heptad repeat `abcdefg` 能直接映射到 helix 表面。alpha helix 每圈约 3.5 residues，7 residues 约两圈，因此 a/d 位点周期性出现在同一侧，形成 hydrophobic stripe，让两条 helix 缠绕配对。

e/g 位点常含 charged residues，用来形成 inter-helical salt bridges，影响 specificity 和 directionality。Leucine zipper 就利用周期性 leucine/hydrophobic residues 形成 dimerization interface，并让相连 DNA-binding domains 正确定位。

## 做题和复习时怎么用

设计 coiled-coil 时先放 a/d hydrophobic residues，再用 e/g charge pattern 决定 parallel/antiparallel 和 partner specificity。Pro 通常破坏 helix，要谨慎使用。
