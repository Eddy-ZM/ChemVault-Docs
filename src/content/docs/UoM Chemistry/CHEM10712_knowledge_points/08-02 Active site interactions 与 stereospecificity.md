# 8.2 Active site interactions 与 stereospecificity

## 详细内容

substrate-active site interactions 可包括：

- van der Waals。
- hydrophobic interactions。
- electrostatic interactions。
- H-bonding。
- 有时形成 covalent enzyme-substrate intermediate。

因为 enzymes 本身由 chiral amino acids 构成，active sites 是 asymmetric 的，所以可选择性结合 stereoisomers，并产生 stereospecific products。

YADH, yeast alcohol dehydrogenase：

- 催化 alcohol 与 NAD+ 之间的 hydride transfer。
- NADH/NADPH 是生物中主要 hydride donors。
- NAD+/NADP+ 是对应 hydride acceptors。
- YADH 中 hydride transfer 的面选择性由 active site geometry 决定。

## 深入讲解

active site specificity 来自多个弱相互作用的总和：shape complementarity、H-bonds、electrostatics、hydrophobic contacts 和有时 covalent intermediate。每个相互作用可能很弱，但正确几何排列能显著偏好某个 substrate 或 stereoisomer。

stereospecificity 来自 chiral active site。即使 substrate 两个面化学上相似，enzyme 也会以固定 orientation 结合，使 hydride transfer、proton abstraction 或 nucleophilic attack 只从一个 face 发生。YADH/NAD+ 例子说明 cofactor 和 substrate 的相对几何决定产物 stereochemistry。

## 做题和复习时怎么用

机制题写 stereospecificity 时，不要只说“enzyme is chiral”。要补充：substrate is bound in one orientation, so only one prochiral face/proton is positioned for reaction.

## 常见误区

- 说酶改变反应平衡，而不是降低 activation barrier。
- 把 Km 简单等同于 Kd，忽略 kinetic model 条件。
