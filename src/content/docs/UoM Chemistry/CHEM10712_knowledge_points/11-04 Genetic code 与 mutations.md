## 详细内容

Genetic code 把 codons 映射到 amino acids。DNA mutation 可导致：

- synonymous mutation：氨基酸不变。
- missense mutation：氨基酸改变。
- nonsense mutation：形成 stop codon。
- frameshift：reading frame 改变，影响更大。

课件中 L23 用序列变化展示：DNA 改变可影响 protein sequence，从而改变 protein function 和 fitness。

## 深入讲解

genetic code 把 nucleotide triplets 转换为 amino acids。因为 codon 是三联体，single-base changes 可能没有影响、改变一个 amino acid、形成 stop codon，或在 insertion/deletion 时改变 reading frame。

mutation 的功能后果取决于位置和性质。active site residue 的 missense mutation 可能严重降低 kcat；surface conservative mutation 可能几乎无影响；regulatory sequence mutation 可改变 expression 而不改变 protein sequence。

## 做题和复习时怎么用

分析突变时先分类：silent, missense, nonsense, frameshift, regulatory。再写可能的 protein/function consequence。
