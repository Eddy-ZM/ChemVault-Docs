# 6.5 Mutarotation 与构象稳定性

## 详细内容

水中 glucose 线性形式很少，cyclic forms 占主要比例。alpha 和 beta anomers 可通过 linear form 互相转化，称为 mutarotation。

要点：

- mutarotation 经 linear form。
- acid/base catalysed。
- chair forms 通常比 boat forms 稳定。
- bulky substituents 倾向 equatorial positions。
- 不同 monosaccharides 的 anomer 稳定性不同。

## 深入讲解

mutarotation 是 alpha 和 beta anomers 通过短暂 linear form 互相转化，导致 optical rotation 随时间变化直到 equilibrium。因为 glycosidic bond 锁定 anomeric carbon，参与 glycosidic bond 的 anomeric centre 通常不能再 mutarotate。

构象稳定性来自 substituent positions。chair conformation 中 bulky groups 偏向 equatorial，因为 axial 会产生 1,3-diaxial repulsion。对 glucose 来说，beta-D-glucopyranose 多个大取代基可处于 equatorial，因此通常更稳定。

## 做题和复习时怎么用

若问哪个 anomer/chair 更稳定，画出 chair 后数 bulky substituents 的 axial/equatorial。不要只背 alpha 或 beta 一定更稳定，不同糖可能不同。

## 常见误区

- 不编号碳就判断 alpha/beta 或 glycosidic linkage。
- 把 D/L 与旋光正负混淆。
