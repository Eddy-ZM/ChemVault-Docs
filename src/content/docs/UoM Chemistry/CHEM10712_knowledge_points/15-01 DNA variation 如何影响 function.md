## 详细内容

DNA sequence 变化可导致 protein sequence 变化。protein sequence 变化可能影响：

- folding。
- active site geometry。
- substrate binding。
- stability。
- expression。
- cellular fitness。

不是所有突变都有益：很多 neutral 或 deleterious。Selection 会富集在特定环境下有 fitness benefit 的 variants。

## 深入讲解

DNA variation 到 function 的路径不是一步完成。突变先改变 nucleotide sequence，再可能改变 mRNA、protein sequence、folding、active-site geometry、stability、expression level 或 regulation，最后才表现为 phenotype/fitness。

同义突变也可能有影响，例如改变 codon usage、mRNA stability 或 translation rate；非编码突变可改变 promoter/RBS 等 regulatory elements。因此 evolution 不只发生在 protein coding changes 上。

## 做题和复习时怎么用

分析一个突变时按层级写：sequence change -> molecular effect -> protein/cellular effect -> fitness consequence。

## 常见误区

- 把工程化写成单一步骤，忽略 selection/screening、flux 和 host fitness。
- 混淆 ALE 的 whole-cell selection 与 directed evolution 的 protein-function screening。
