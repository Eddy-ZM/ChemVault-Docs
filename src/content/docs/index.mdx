## ChemVault Knowledge System

ChemVault Docs is the documentation and knowledge base for the ChemVault ecosystem.

It connects chemistry learning, laboratory preparation, project documentation, reagent references and future research tools into one structured archive.

  <Card title="Chemistry Notes" icon="seti:folder">

## Current Modules

- **ChemVault Core** — the main chemistry research platform.
- **ChemVault Mail** — the email system for the ChemVault domain.
- **ChemVault Docs** — the documentation and knowledge base.
- **ChemVault Files** — a future resource and download centre.
