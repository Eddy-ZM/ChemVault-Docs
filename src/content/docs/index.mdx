# ChemVault 使用手册

这里是 ChemVault 的教程型文档入口。它把 `E:\CV_suite` 中的主站、中文站、Extract、Files、Mail、Notification、User Center、Molecule Studio 和 Apple App 功能整理成面向用户的说明书。

## 推荐阅读顺序

1. [产品地图](/manual/product-map/)：先了解每个站点、App 和服务的职责。
2. [快速开始](/manual/getting-started/)：按学生、研究者、管理员或开发者身份选择路径。
3. [主站与中文站](/manual/chemvault-website/)：学习如何浏览公开内容和化学记录。
4. [ChemVault Extract](/manual/extract/)：把论文、报告和仪器文件变成可审核数据。
5. [术语表](/manual/glossary/)：查阅 SMILES、PDB、Webhook、RBAC、Review Flow 等专业词。
