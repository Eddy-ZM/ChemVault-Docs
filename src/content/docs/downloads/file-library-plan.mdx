ChemVault Docs can act as the public index for future downloadable resources.

This page defines the filing rules before the downloads area becomes large.

## File categories

Planned file categories:

- lecture summaries
- reagent sheets
- reaction mechanism templates
- lab report templates
- project screenshots and diagrams
- release notes
- archived course resources

## Naming convention

Use names that remain understandable outside the website.

Recommended pattern:

```text
chemvault-<workspace>-<topic>-<version-or-date>.<extension>
```

Examples:

```text
chemvault-reagents-nabh4-passport-v1.pdf
chemvault-chem10712-exam-patterns-2026-06.pdf
chemvault-lab-tlc-monitoring-template-v1.docx
```

## Metadata checklist

Every downloadable file should have:

- title
- short description
- owner or maintainer
- source page
- version or date
- file type
- intended audience
- update status

## Link policy

Each file should be linked from a context page, not only from a file list.

For example, a TLC template should be linked from the TLC guide, and a reagent sheet should be linked from the matching reagent passport.
