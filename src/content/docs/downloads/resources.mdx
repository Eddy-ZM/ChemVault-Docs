## Release states

| State | Meaning | Publish rule |
| --- | --- | --- |
| Draft | Content exists but needs review, metadata or route context. | Keep internal or clearly labelled. |
| Candidate | The resource has context, owner, title, date and intended audience. | Link from a relevant context page. |
| Released | The resource is stable enough for public use. | Add to this library and record in the changelog. |
| Archived | The resource is superseded or retained for reference. | Keep the archive reason visible. |

## Metadata checklist

- Title and short description
- Owner or maintainer
- Version or date
- Intended audience
- Related documentation route
- Release state
- File type and expected use

## Current status

The file catalogue is intentionally conservative. The public route exists now so future downloads can be added without changing the site architecture later.
